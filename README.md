# My NixOS dot files

Flake based configuration of NixOS.

## Features

- Dank Material Shell + Niri;
- Roboto Condensed && Monaspace Neon NF;
- Alacritty (zsh + starship);
- SOPS secrets (fetching from BitWarden).
